# SharkNET Blockchain
Мой собственный самописный блокчейн на базе PoW, с комиссиями, майнингом и удобным CLI. Для удобства использования я запустил bootstraper и node на одном своём сервере (sharknet.g0shark.ru), благодаря чему сеть функционирует 24\7 (пока я продливаю сервера)

## Wallets on nicknames
в отличии от других сетей, в свою я напрямую встроил систему никнеймов. Вместо того, чтобы писать длинный и не удобный адрес, в транзакциях пишется простой ник - благодаря этому, переводить валюту становится гораздо удобнее, а вместе с комментариями к переводу (до 128 символов) вы можете отправлять простые сообщения вместе с переводом

## Taxes
Для каждого перевода считается коммисия в виде 10% от транзакции, которые переходят майнеру, что замайнил блок с транзакцией. Даже так, базово майнер получает 10 валюты за каждый блок

## Difficulty
Сложность пересчитывается каждые 5 блоков, в основном стараясь прийти к скорости 30 минут\блок

# Build
1. Склонируйте репозиторий
2. Забилдите проект ```dotnet build && dotnet run```

---
# Usage
1. Скачайте CLI интерфейс с [последнего релиза](https://github.com/G0Shark/SharkNetBlockchain/releases/latest)
2. Распакуйте ZIP в любом удобном месте
3. Запустите Blockchain.exe
4. Создайте кошелёк с никнеймом

В будущем ваш кошелёк будет хранится в файле wallet_(port).json, **НЕ СКИДЫВАЙТЕ ЭТОТ ФАЙЛ!**, благодаря нему можно получить доступ к вашему кошельку. Переносите файл к любым программам из релиза (Нарпимер, к чистому майнеру), и он будет исопльзовать этот кошелёк как основной

# Avaliable commands 
1. ```wallet``` - показывает информацию о вашем кошельке
2. ```balances``` - показывает балансы других участников сети
3. ```peers``` - показывает участников сети
4. ```chain``` - отображает цепочку транзакций в консоль
5. ```tx <to> <amount> [message]``` - отправить деньги другому участнику по никнейму
6. ```mine``` - замайнить блок, берёт 10 транзакций
7. ```minegpu``` - замайнить блок, использует видеокарту, берёт 10 транзакций
8. ```exit``` - выйти с программы
